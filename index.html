Keyboard Shortcuts & Modifiers

  • P - Pencil Tool
  • L - Line Tool
  • R - Rectangle Tool
  • C - Ellipse Tool
  • S - Selection Tool
  • E - Eraser Tool
  • F - Fill Tool
  • A - Airbrush Tool
  • T - Text Tool
  • Ctrl+C - Copy Selection
  • Ctrl+X - Cut Selection
  • Ctrl+V - Paste Selection
  • Ctrl+Shift+O - Crop to Selection
  • H or F1 - Toggle Help
  • Shift - Constrain proportions
  • Alt - Draw from center
